
# Exploratory Analysis of Rainfall Data in India for Agriculture

## Description
This project analyzes rainfall data and predicts whether conditions are suitable for crop cultivation using Machine Learning.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Flask

## Steps to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
